Vue JS Cookbook

A collection of best practices for VueJS 2, shown through a fully functional enterprise ready app

See on Github

Why

Because TodoMVC just ain't cutting it.

Our goal is to build a full scale application, that's structured for large scale applications and has recipes for all the major interactions you'd need in an application. Don't keep reinventing the wheel, start with a recipe!

What

We plan to build a working example application that includes all of the following behaviors and interactions.

  • Pagination
  • Authentication
  • Toast Messages
  • Page Loading
  • Component Loading
  • Backend API Integration
  • And much more...

Want to Help?

Create an issue on Github if you want to help contribute!
